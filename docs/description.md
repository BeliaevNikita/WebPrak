Описание страниц:
![Навигация страниц]
У каждой страницы шапка состоит из:
* Кнопка Главная
* Кнопка Каталог
* Кнопка для перехода в Личный кабинет/ перехода на страницу для Входа/Регистрации

Главная страница:
* Кнопка перехода в каталог с пояснением, что заказать книги можно через каталог
* Кнопка перехода в Личный Кабинет с пояснением, что изменить информацию о себе, посмотреть свои заказы или оформить возврат, если с момента доставки заказа прошло не больше недели
* Контанкты для связи

Каталог книг:
* Список 50 самых популярных книг
* Кнопка показать ещё
* При нажатии на запись о книге происходит переход на страницу описание книги (сверху в каталоге будет пояснение об этом)
* Поиск и фильтры по названию, автору, жанру, цене, дате издания

Страница книги:
* Название, описание, характеристики (автор, жанр, цена, дата издания)
* Фото книги
* Кнопка для заказа
* Кнопка добавить в корзину (для пользоватлея есть корзина, которая находится в личном кабинете и которая является заказом со статусом draft и видна в списке заказов)

Личный кабинет:
* Форма с личными данными, которые можно менять и кнопка сохранить изменения
* Список заказов (первым отдельно от остальных идет корзина), для каждого заказа можно нажать на запись, чтобы перейти на старницу заказа

Станица заказа:
* Информация о позициях
* Справа от каждой позиции есть чекбокс
* Если это корзина, то есть кнопка Оформить заказ, и можно убрать галочку от товаров, которые не войдут в итоговый заказ
* Если это уже полученный заказ, то есть кнопка оформить возврат, и можно выставить галочки для позиций, которые подлежат возврату
* Подсчитывается итоговая стоимость

Следующие страницы доступны только работникам магазина:

Панель управления работника:
* Кнопка Список книг
* Кнопка список заказов

Страница список книг:
* Содержит те же фильтры, что и Каталог
* Каждая запись содержит характеристики книги
* Можно менять характеристики книги, добавлять новые и удалять

Страница список заказов:
* Записи о всех заказх клиентов
* Можно менять статус заказа, есть фильтры по статусу, дате, клиенту

Описание use cases/сценариев использования:

Клиент:
* Клиент заходит на сайт и попадает на главную страницу сайта. Оттуда он может попасть в форму входа/регистрации, зайти в каталог книг, смотреть по поиску и фильтрам доступные в книги в магазине, или перейти в личный кабинет.
* В катологе при нажатии на запись о книге, клиент попадает на страницу с информацией о книге и может добавить ее в корзину, которая находится в Личном кабинете. Также клиент может искать нужные себе книги с помощью фильтров.
* В Личном кабинете клиент может изменить свою данные, увидеть список заказов и посмотреть подробности о конкретном заказе, перейдя на страницу Подробности о заказе при нажатии на запись о заказе. На этой странице он может оформить возврат для части позиций, поставив галочки в чекбоксы, соответствуюшие этим товарам, и нажав кнопку Оформить возврат.
* Также в Личном кабинете клиент может перейти в Корзину. Это перенесет его на страницу Подробности о заказе, и он сможет оформить заказ, нажав кнопку Оформить заказ и поставив галочки в чекбоксы для позиций, которые в итоге он хочет взять

Работник:
* После авторизации попадает на страницу Панель управления работника. При нажатии на кнопку список книг попадает на страницу список книг, где может добавлять новые книги, изменять информацию о существующих и удалять их. При нажатии на кнопку список заказов попадает на страницу список заказов, где может менять статус заказов (например, одобрение возврата, сообщение о готовности заказа и доставке заказа)
